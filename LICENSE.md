## Which license applies?

When in doubt:

- If it's text the writer wrote (prose, headings, captions, lists,
  explanations) → **CC BY 4.0**
- If it's code in a fenced code block (Rust, Python, TypeScript, shell,
  config files, etc.) → **MIT OR Apache-2.0**
- If a specific post or file declares a different license, that declaration
  governs that specific work.

## Attribution example

When citing prose from this site, the preferred attribution format is:

> SHA888 (YYYY). *Title of the post*. MAIS Log. https://sha888.id/<path>

For notes specifically, citing the commit hash is preferred over date,
because notes are editable and the commit hash uniquely identifies the
version you read:

> SHA888. *Title of the note*. MAIS Log, commit `<short-hash>`.
> https://github.com/SHA888/mais-log/commit/<full-hash>
